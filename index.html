Running the demonstration¶

In order to download the program and run its demonstration, use the following command (scroll to right if need be):

git clone git@github.com:reof-mas/reof-mas-src.git && cd reof-mas-src && pyvenv-3.5 py_env && . py_env/bin/activate && pip install -r requirements.txt && cd py_env && python demo.py
